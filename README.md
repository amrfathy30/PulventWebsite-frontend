Pulvent - Technology & Marketing Website
A modern, responsive landing page for Pulvent, a technology, marketing, and consulting company.

Features
Bilingual Support: Full English and Arabic language support
RTL Layout: Automatic Right-to-Left layout when Arabic is selected
Responsive Design: Optimized for mobile, tablet, and desktop screens
Language Persistence: User's language preference is saved across sessions
Tech Stack
React 18 with TypeScript
Vite for fast development and building
Tailwind CSS for styling
shadcn/ui for UI components


Fonts
English: Montserrat
Arabic: Cairo


Sections
Hero Section - Main banner with navigation
Services Section - Company services showcase
About Section - Company information
Features Section - Core philosophy and values
Contact Section - Contact form
Footer - Links and social media


Language Toggle
Click the language button in the header to switch between English and Arabic. The entire layout will automatically adjust for RTL when Arabic is selected.
